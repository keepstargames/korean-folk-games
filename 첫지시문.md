# 클로드 코드 첫 지시문 (복사해서 붙여넣으세요)

아래 내용을 클로드 코드 채팅에 그대로 붙여넣으면 됩니다.

---

이 폴더에는 완성된 웹게임 yutnori.html과 프로젝트 규칙 CLAUDE.md가 있어.
여러 게임을 담을 포트폴리오 사이트의 기반을 만들어줘. 순서대로:

1. 먼저 전체 계획을 보여주고 내 승인을 받은 뒤 진행해.

2. git 저장소를 초기화하고 이런 구조로 정리해:
   - games/yutnori/index.html  (지금 yutnori.html 그대로, 게임 로직 수정 금지)
   - index.html  (허브 페이지: 게임 목록을 보여주는 랜딩. yutnori.html과 같은
     한지·먹 디자인, 한/영 전환 포함. 지금은 윷놀이 하나만 카드로 표시)
   - shared/  (앞으로 게임들이 공유할 모듈 자리. 이번엔 yutnori에서 코드를
     빼내지 말고 폴더와 README만 만들어)

3. 로컬에서 확인할 수 있게 간단한 실행 방법을 알려줘 (예: npx serve).
   내가 브라우저로 윷놀이가 그대로 잘 도는지 확인할게.

4. 내가 확인해주면 GitHub에 올리는 것과 Cloudflare Pages 무료 배포를
   단계별로 안내해줘. 계정 가입과 버튼 클릭은 내가 직접 할 테니,
   네가 할 수 있는 부분(설정 파일, 배포 명령)은 네가 하고
   내가 해야 하는 부분은 화면에 뭐가 보일지까지 구체적으로 알려줘.

한 단계 끝날 때마다 멈추고 다음으로 넘어가도 되는지 물어봐.
