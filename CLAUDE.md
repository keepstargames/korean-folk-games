# 한국 전통놀이 웹게임 포트폴리오

## 프로젝트 목표
- 한국 전통놀이(윷놀이, 고누, 알까기, 칠교놀이 등)를 웹게임으로 만들어
  자체 사이트, itch.io, HTML5 게임 포털(CrazyGames, GameDistribution 등)에 배포한다.
- 수익 모델: 광고. 목표는 월 $100-200.
- 첫 게임 yutnori.html은 완성된 상태다. 게임플레이 로직과 밸런스를 임의로 바꾸지 말 것.

## 모든 게임의 공통 규칙
- 한국어/영어 이중언어 지원 (언어 전환 버튼, i18n 딕셔너리 방식 — yutnori.html 참고)
- 사운드는 WebAudio로 코드에서 합성. 외부 오디오/이미지 파일 사용 금지 (저작권 제로 유지)
- 폰트: Google Fonts(Song Myung, Gowun Dodum) + 한국어 시스템 폰트 폴백 필수
- localStorage / sessionStorage 사용 금지 (일부 배포 환경에서 차단됨). 세션 내 메모리만 사용
- 모바일 우선: 세로 화면에서 조작 가능해야 함, 터치 타깃 44px 이상
- prefers-reduced-motion 존중

## 작업 방식
- 파일을 삭제하거나 크게 재구성하기 전에 반드시 계획을 먼저 보여주고 승인받을 것
- 각 작업 후 게임이 실제로 동작하는지 확인할 것 (최소한 문법 체크와 로컬 서버 실행)
- git 커밋은 작은 단위로, 한국어 커밋 메시지 사용
- 배포 대상: Cloudflare Pages (무료), itch.io (butler CLI), 포털별 SDK 빌드는 추후

## 하지 말 것
- 게임 규칙/확률/AI 밸런스 임의 변경 금지 (요청받은 경우만)
- 유료 서비스 가입이 필요한 도구 제안 금지 (무료 티어만)
- 화투/고스톱류 게임 제작 금지 (사행성·저작권 이슈)
